============================================
如何替换
============================================
1. 将你的作品图片命名为上述文件名之一
2. 放入此文件夹
3. 刷新浏览器即可看到效果

============================================
推荐规格
============================================
尺寸: 600 × 440 px（或 3:2 比例）
格式: JPG 或 WebP
大小: 每张建议 < 200KB

============================================
关键 CSS 选择器
============================================
.work-preview         → 控制预览窗样式
width: 300px;         → 显示宽度（CSS 会自动缩放）
