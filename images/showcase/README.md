# images/showcase/ — 作品集大图展示区（视差面板）

这些图片用于 Collection Highlights 展示区，每块面板覆盖整个屏幕 70% 的高度，
带有视差滚动效果（背景缓慢移动）。

============================================
文件列表（共 3 个）
============================================

chromatic-bg.jpg      → Chromatic Explorations 系列背景
algorithmic-bg.jpg    → Algorithmic Landscapes 系列背景
lightspace-bg.jpg     → Light & Space 系列背景

============================================
如何替换
============================================
1. 将你的作品图片命名为上述文件名之一
2. 放入此文件夹
3. 刷新浏览器即可看到效果

============================================
推荐规格
============================================
尺寸: 1920 × 1200 px（或 16:10 比例）
格式: JPG 或 WebP
大小: 每张建议 < 500KB
注意: 背景图会被放大 1.08 倍并应用视差偏移，务必留足边缘

============================================
关键 CSS 选择器
============================================
.showcase-item        → 每块面板容器（height: 70vh）
.showcase-bg          → 背景图层（inset: -20%，覆盖溢出）
.showcase-overlay     → 渐变遮罩（底部加深）

============================================
HTML 位置
============================================
在 index.html 中搜索 "showcase-bg" 找到对应行，
修改 background-image:url(...) 中的路径即可。
